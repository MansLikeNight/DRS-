Logo Files:
-----------
Place your Leos Investments Ltd logo file here with the name:
- leos-logo.png (recommended size: 200x60 pixels or similar aspect ratio)

The logo will be displayed in:
1. Navigation bar (white version, height: 40px)
2. Django admin interface
3. Login/registration pages

If you don't have a logo yet, you can use the placeholder logo.png or create a simple one.

For best results:
- Use PNG format with transparent background
- Keep the logo simple and readable at small sizes
- Consider creating a white version for dark backgrounds
